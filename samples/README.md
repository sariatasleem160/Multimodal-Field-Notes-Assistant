# Samples

Place evaluation inputs here.

## Images (`images/`)

Run `python scripts/generate_sample_images.py` to create five PNGs:

| File | Mode |
|------|------|
| `receipt_sample.png` | Receipt extraction |
| `whiteboard_sample.png` | Whiteboard bullets + actions |
| `chart_sample.png` | Bar chart |
| `equipment_sample.png` | Field equipment photo |
| `screenshot_sample.png` | UI screenshot |

You can also add your own JPG/PNG photos.

## Audio (`audio/`)

Run `python scripts/generate_sample_audio.py` (needs API key) to create:

| File | Content |
|------|---------|
| `meeting_notes.mp3` | Standup with decisions and action items |
| `voice_memo.mp3` | Field site voice note |

Or record your own `.mp3` / `.wav` files named `meeting_notes.*` and `voice_memo.*`.
